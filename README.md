# CodeAlpha_EDA
# Customer Shopping Behavior Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a customer shopping behavior dataset. The objective is to understand customer purchasing patterns, identify trends, and generate business insights that can support strategic decision-making.

## Dataset Information

* Total Records: 3,900
* Total Features: 18
* Data Includes:

  * Customer demographics
  * Purchase information
  * Product categories
  * Subscription status
  * Shipping preferences
  * Review ratings

## Tools Used

* Python
* Pandas
* Jupyter Notebook

## Business Questions and Insights

### 1. Which gender generates more revenue?

Female customers generate slightly higher total revenue compared to male customers.

### 2. Which product category is purchased most frequently?

The Clothing category records the highest number of purchases.

### 3. Do subscribers spend more than non-subscribers?

Subscribers show a higher average spending pattern than non-subscribers.

### 4. Which shipping type is associated with higher spending?

Customers using Express Shipping tend to spend more on average.

### 5. Which age group spends the most?

Age-group analysis was performed to identify high-value customer segments.

## Conclusion

The analysis highlights valuable customer behavior patterns that can help businesses improve marketing strategies, customer retention, and revenue generation.
